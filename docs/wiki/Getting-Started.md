# Getting Started

This project is a pre-alpha Lone Wolf Action Assistant Redux rebuild. Book 1 is playable enough for local testing, but it is not a packaged public release yet.

To run the local shell:

```powershell
.\Launch-LoneWolfRedux.ps1
```

Then open:

```text
http://localhost:8797/assistant.html
```

For the first build pass, use the local Book 1 files:

```text
books\lw\01fftd
```

Do not commit the book files.

## Playing Book 1

Start a new Book 1 character from the assistant. The current Book 1 sheet supports Combat Skill, Endurance, Kai Disciplines, Weapons, Backpack Items, Special Items, Meals, and Gold Crowns.

Use the assistant buttons when the current section offers them:

- route checks for Kai Discipline, item, Gold, or END branches
- loot buttons for optional items
- roll helpers for random-number sections
- combat presets for audited fights
- death recovery controls when a run ends badly

For practical route advice, see [Book 1 Strategy Guide](Book-1-Strategy-Guide).
