# Full Campaign Story Run

This page will become the clean campaign route as more Lone Wolf books are added.

For now, it records the Book 1 route we use when we want a steady story clear to Holmgard. It is not the only winning route, and it is not meant to show every branch.

## Before You Start

For a smooth Book 1 run, the friendly first-clear Kai Discipline set is:

- Camouflage
- Sixth Sense
- Tracking
- Hunting
- Healing

That set gives you route awareness, food safety, and steady Endurance recovery.

## Book 1: Flight from the Dark

This route reaches Holmgard:

```text
1 -> 141 -> 56 -> 222 -> 252 -> 70 -> 157 -> 30 -> 261 -> 264 -> 6 -> 200 -> 168 -> 64 -> 16 -> 192 -> 171 -> 303 -> 237 -> 265 -> 142 -> 135 -> 223 -> 75 -> 163 -> 321 -> 273 -> 51 -> 288 -> 129 -> 3 -> 196 -> 332 -> 350
```

The route uses ordinary book links, route checks, and a section 237 random-number branch.

For advice on why this is a comfortable first-clear route, see the [Book 1 Strategy Guide](Book-1-Strategy-Guide).
