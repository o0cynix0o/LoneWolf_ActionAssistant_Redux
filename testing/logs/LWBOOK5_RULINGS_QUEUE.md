# LW Book 5 Rulings Queue

Scope: user rulings and implementation assumptions for Book 5, *Shadow on the Sand*.

This file keeps the human calls close to the technical audit so the next pass can tell which behavior came from the book scan and which behavior came from table rulings.

## Approved Rulings

- Backpack Items carry forward from Book 4 into Book 5 unless a later book rule explicitly removes them.
- Book 5 setup adds one new Kai Discipline.
- Book 5 Gold uses the normal hard 50 Gold Crown cap. Gold that would exceed the cap is not picked up.
- Book 5 starting equipment allows up to four different choices from the available list.
- Special Item safekeeping is a real campaign mechanic going forward. Items placed in safekeeping leave the active Action Chart, are unavailable during the current adventure, are protected from ordinary loss/confiscation, and remain tracked for later books.
- Safekeeping is place-aware. Book 5 currently records the place as `Kai Monastery`, and future books can add their own safekeeping place labels without mixing that state into temporary confiscation.
- Book 5 palace confiscation is temporary gear storage, separate from safekeeping.
- If the book specifically says Hunting cannot help, Meal automation eats from the Backpack or applies the listed END loss when no Meal is available.
- Book 5 terminal failure states are recorded like death/failure for recovery and history, even when Lone Wolf is alive but the mission has failed.
- Book 5 Blood poisoning is a continuing section-entry effect until cured by the configured cure item/effect.
- The Sommerswerd loss in Book 5 is tracked as a temporary story state and restored by the finale helper.
