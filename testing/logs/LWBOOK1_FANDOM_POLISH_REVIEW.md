# Book 1 Fandom Polish Review

Scope: Flight from the Dark polish pass using fan-wiki pages as a secondary context source.

Canonical rule source remains the local Project Aon Book 1 files. The fan wiki is useful for route context, item importance, and friendly guide language only.

## Useful Signals

- Banedon is a rare, lore-important meeting and depends on the right early route/Discipline context.
- Crystal Star Pendant is already a Book 1 achievement target and should be called out as a replay-friendly discovery.
- The walkthrough reinforces that Book 1 works best as a survival run first and a branch cleanup run later.
- Weaponskill is less reliable in Book 1 than the safer route-reading and survival Disciplines.

## Decisions

- Keep Book 1 rules and automation unchanged.
- Strengthen public guide language around Banedon and the Crystal Star Pendant.
- Add Crystal Star Pendant to the inventory key-item badge list.
- No new Book 1 achievement needed; `lw1_crystal_star_pendant` already covers the useful replay target.

## Implemented In This Pass

- Inventory key-item note for Crystal Star Pendant.
- Book 1 guide can keep its existing structure; later copy edits may add a Banedon subsection if we want more flavor.
