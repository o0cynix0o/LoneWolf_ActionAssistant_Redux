Install The Books

Lone Wolf Action Assistant Redux does not include the Project Aon book files. Download the standard multi-page HTML ZIPs from Project Aon for your own personal use, then extract them into this app.

Home Assistant Project Aon License

Download Links

1. Flight from the DarkOpen the Project Aon page and choose the standard ZIP.

Folder Layout

Extract the Book 1 ZIP so this file exists:

books\lw\01fftd\title.htm

PowerShell Example

New-Item -ItemType Directory -Force .\books\lw

Expand-Archive "$env:USERPROFILE\Downloads\01fftd.zip" -DestinationPath .\books\lw -Force

See docs\INSTALL_PROJECT_AON_BOOKS.md for the full walkthrough.
